Abri proyecto en VSCODE
1. npm install -g firebase-tools (sino est√° instalado/primera vez)
2. firebase login

	3.2 No instalar las dependencias del lenguaje seleccionado (No)
4. Entrar a la carpeta functions: cd functions
5. Activar el entorno virtual:

6. python -m pip install --upgrade pip
7. npm install firebase-admin firebase-functions
8. pip install firebase-admin
9. pip install firebase_functions
10. pip install Flask
11. Modificar el json:
	11. Para programar
12. Iniciar en emulador Firebase suit (cd carpeta raiz)
	12.1 firebase emulators:start --only functions (solo las funciones)
	12.2 todo completo: firebase emulators:start

13. Hacer deploy (consultar con GEMINI, para subir a la nube).
